# Wenak Legal Documents

Public hosting for Wenak's Privacy Policy and Terms & Conditions, published
via GitHub Pages.

## Structure

- `privacy-policy/` — Privacy Policy (Arabic default at index, English at en.html)
- `terms/` — Terms & Conditions (Arabic default at index, English at en.html)
- `index.md` — landing page linking to both
- `_layouts/default.html` — shared page layout (handles RTL/LTR)
- `_config.yml` — Jekyll config

## Live site

https://taha-elkholy.github.io/wenak-legal/

The Arabic version of each document is the legally authoritative text.
